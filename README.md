# Investment-accounts
This project is a JavaScript investment account simulator that uses an array of account balances to represent different users. It allows users to deposit money, withdraw funds, count accounts with balances under $2000, apply donations to lower-balance accounts, and simulate a hacker attack that steals money from all accounts. The program also displays the account balances visually in a dynamic bar graph that updates continuously. It uses arrays, loops, functions, conditionals, random number generation, and DOM manipulation to create an interactive financial simulation.
